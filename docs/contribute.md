# Contribute

There are many ways to contribute to the KD project, ranging from working on documentation
and writing unit tests to implementing KD for a new language / platform:

* File issues and feature requests in our [issue and feature tracker](https://github.com/kixi-io/Ki.Docs/issues)
* [Fix issues](https://github.com/kixi-io/Ki.Docs/issues)
* Write or edit documentation
* Contribute [unit tests](https://github.com/kixi-io/Ki.KD-JVM/tree/master/src/test/kotlin/io/kixi/kd)
* Submit a pull request
* Port KD to a new language / platform

## Implementations
[KD-JVM release 2.2.0](https://github.com/kixi-io/Ki.KD-JVM) (Kotlin, Java, Scala, etc.) is feature
complete. We have leads spinning up implementations for Go and Swift. Our team is actively looking for leads to take on Python, TypeScript, Rust, .NET (C#) and Ruby.
